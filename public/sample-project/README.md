# Sample Project — QA Workshop Practice

This project contains **intentional bugs** for practice during the workshop.

## Files

- `auth.js` — Authentication module (register, login, reset password, profile)
- `api.js` — API endpoints (search, profile update, payments, file upload)

## Your Tasks

### Task 1: Generate Test Cases
Run `/test-cases` on the registration page functionality:
```
/test-cases Registration page with fields: name, email, password, confirm password
```

### Task 2: Find Bugs
Open `auth.js` and use your AI agent to find bugs:
```
/bug-report [describe what you found in auth.js]
```

## Hints
- `auth.js` contains **15 intentional bugs** across 4 functions
- `api.js` contains **15 intentional bugs** across 4 functions
- Look for: validation gaps, security issues, logic errors, edge cases

Good luck, cadet.
